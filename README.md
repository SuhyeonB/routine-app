# 🔁 Routine App

매일 반복되는 루틴을 관리하는 개인용 웹 앱

## 기술 스택

- Kotlin + Spring Boot 3
- Thymeleaf + Tailwind CSS
- Spring Data JPA
- PostgreSQL
- 배포: Railway

## 기능

- 루틴 추가 / 수정 / 삭제
- 반복 설정 (매일 / 매주 요일 / N일마다)
- 카테고리 관리 (색상 지정)
- 오늘의 루틴 체크
- 주간 / 월간 통계

## 실행 방법

```bash
./gradlew bootRun
```

로컬 실행 후 `http://localhost:8080` 접속
